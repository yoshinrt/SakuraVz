[ 初めに ]

　オープンソースのテキストエディタ、"サクラエディタ"のPukiwiki用の強調定義ファイル群
です。Pukiwikiを編集する時に私はエディタ上で書いてからポストしてますが素のエディタ
表記ではどうしても不便なので即席で書いてみました。

--------------------------------------------------------------------------------------

[ 使い方 ]
　サクラエディタに設定ファイル群を読み込ませて適用させれば問題ない筈です。読み込ませ方
等はエディタのヘルプを見てください。
　尚、作成者はBregexp.dllではなく、bregonig.dllを使用しています。よって正規表現周りで
問題が発生しうるかもしれません(多分大丈夫と思いますが)。

--------------------------------------------------------------------------------------

[ アーカイブ内容 ]
　pukiwiki.col   カラー定義ファイル
　pukiwiki.kwd   強調キーワードファイル
　pukiwiki.rkw   正規表現キーワードファイル
　readme.txt     コレ
