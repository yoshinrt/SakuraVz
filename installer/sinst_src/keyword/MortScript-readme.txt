☆サクラエディタでMortScriptを編集するための各種定義ファイルです。
　　　　　　　　　　　　(無保証・無サポート好きに使って Copy Left りーや)

同梱の各ファイルは次の通り
Mortscript-readme.txt このファイルです
MortScript.kwd	キーワード定義ファイル (フロー制御構造)
MortScript2.kwd	キーワード定義ファイル２ (コマンドと関数)
MortScript.rkw	正規表現キーワード定義ファイル
MortScript.khp	キーワードヘルプの辞書ファイル
MortScript.hkn	入力補完機能の単語ファイル
MortScript.rule	アウトライン解析のルールファイル
MortScript.col	色設定ファイル

マニュアル v4.2 要訳などを元にしています。作者さま翻訳者さまに大謝です。

■この各種定義ファイルの旧バージョンからの置き換えの方へ
旧い各種定義ファイルにすべて（上書きで）置き換えて下さい。以下の使い方
の説明に従って、新規導入時と同様に一通り見直して視るのをオススメします。
※強調キーワードのセット名 「MortScript2」の中に以下の古いコマンド名が入
っていたら、個別に削除しておいて下さい。
            GetVolume   Rnd   WriteStatusList
※特にインポートするところでのインポートのやり直しを忘れないようにご注意

■サクラエディタ本体より先にこれを入手してしまった方へ
これは、日本語版 Windows95/98/98SE/Me/NT4.0/2000/XP/2003/Vista で動く
オープンソースで高機能でフリーのテキストエディタであるサクラエディタに
適用させる MortScript 編集支援のためのファイル群です。
サクラエディタ本体は、インターネットを検索の上で、一次配布場所から取得
して下さい。

※注１【重要】英大文字小文字の同一視についての指針
　MortScript は言語仕様として、コマンド名や変数名等で大／小文字の区別を
しません。これを同一視させているのは、手軽なスクリプト言語のひとつの答え
だと思いますが、美しさが損なわれ可読性を悪くするという側面も無視出来ない。
　試行錯誤のすえに、自分が行うスクリプティングでは、大／小文字を区別とし
て見易い（≒高品質）を目指すことも出来ながら、一方で、他人の作ったスクリ
プトなどを読み解く時には言語仕様通りに色分け表示できるようにしました。
※これらの設定は、個人の趣味の部分もありますので、お好みで変えて下さい。


■使い方
同梱のファイルをサクラエディタのインストール先(C:\Program Files\sakura等)
の下の「keyword」フォルダに、すべてまとめてコピーしてください。

以下、サクラエディタ上での設定例です。
メニューの「設定(&O)」から「共通設定(&C)」や「タイプ別設定一覧(&L)」など
を選んで指定します。

○共通設定 『強調キーワード』タブ
・「MortScript」と「MortScript2」の２つの「セット名」で、強調キーワード
　の「セット追加」をしてください。
それぞれ、MortScript.kwdとMortScript2.kwdを[インポート]します。
・「英大文字小文字区別のチェック(&C)」は、付けずに同一視。（注１）
必ず「ＯＫ」ボタンで閉じます。

○タイプ別設定一覧
「設定nn」の未使用のタイプか、それが残ってなければ使わなそうなタイプを選
んで、「設定変更(&S)」ボタンを押して、「タイプ別設定」を表示させ以下の設
定を行います。

○タイプ別設定 『スクリーン』タブ
・設定の名前→ MortScript
・ファイル拡張子→ mscr,mortrun
・レイアウトTAB幅→  4
 （タブ幅はお好きに調整）
・アウトライン解析方法でルールファイルを選択し同梱のMortScript.rule
　を指定します。
必ず「ＯＫ」ボタンで閉じます。

○タイプ別設定 『カラー』タブ
同梱の色設定ファイルMortScript.colを[インポート]してください。
「強調キーワード&2〜10」ボタンを押して、強調キーワード1と2に、それぞれ
上記の共通設定で作った「MortScript」と「MortScript2」をセット。
その他を次のようにします。
<コメントスタイル>
  行型  #  （行型の一番上に # を指定）
  行型  //  (行型の二番目に # を指定）※ MortScript 4.3b2
  行型  ;  （行型の三番目に # を指定）※ MortScript 4.3b2
<エスケープ>
  ・PL/SQL風: """"  (ラジオボタンでこちらを選択)
必ず「ＯＫ」ボタンで閉じます。

想定上の色分けは次の通り
　強調キーワード1 … フロー制御構造
　強調キーワード2 … コマンド・関数など
　強調キーワード3 … 演算子
　強調キーワード4 … 定義済み変数など
　強調キーワード5 … 古い表記方法となり互換性からありでも推奨されないモノ
　強調キーワード6 … 開発途上のベータ版のみで提供されるモノ
※色などは、お好みで変えていただくのも良いと思います。

○タイプ別設定 『支援』タブ
・入力補完機能  単語ファイル に同梱のMortScript.hknを指定
・「英大文字小文字を同一視する(&I)」は、チェックしない。（注１）
・「編集中のファイルから候補を探す(&F)」のチェックは、可読性を重視して
　わかり易く長めの変数名を使う人には、オススメです。
必ず「ＯＫ」ボタンで閉じます。
※別途、関連品のHTML Help生成ツールでMortScript.chmを生成して、
　外部ヘルプの設定(&L)で設定すると便利です。

○タイプ別設定 『正規表現キーワード』タブ
「正規表現キーワードを使用する」をチェックして、同梱のMortScript.rkwを
インポートしてください。強調表示の補完などを行っています。
記号以外は最後に i (大／小文字同一視)オプションを付けています。（注１）
　（詳細内容の意味合いは、定義ファイル内にコメント記述）
必ず「ＯＫ」ボタンで閉じます。

○タイプ別設定 『キーワードヘルプ』タブ
「キーワードヘルプ機能を使う」をチェックして
辞書ファイルに同梱のMortScript.khpを選んで、「挿入(&S)」ボタンを押して
ください。
※ここは、インポートではありません。
必ず「ＯＫ」ボタンで閉じます。

○サクラエディタを一度、完全に終了させる
「ファイル(F)」メニューから「サクラエディタの全終了(X)」を選んで、終
了させて、念のために、これまでの変更が設定ファイルなどに反映させます。

サクラエディタでMortScriptを書こうという人に、たぶんこんな余計な説明は
不要だったのではとも思われます。
サクラエディタのヘルプ上に情報は揃ってるので、必要に合わせて参照の上で
お好みでアレンジして、ご利用下さい。

■その他
改良が頻度高く進められているMortScriptで、保守には不安が大きいです。
まぁ、使い物にはなるかという完成度で、改善・改良は大歓迎です。
より良いものが出来ましたらシェアさせてください。
